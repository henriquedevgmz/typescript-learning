## 👋 Sobre este repositório

Este repositório é meu **caderno de estudos de TypeScript** — um espaço pessoal onde pratico, experimento e consolido conceitos da linguagem enquanto avanço no meu caminho como desenvolvedor.

> Nada aqui é perfeito. Mas tudo aqui é real.

Cada módulo representa um passo na curva de aprendizado. Se você chegou aqui, seja bem-vindo — e fique à vontade para explorar.

O conteúdo é baseado na **[Documentação Oficial do TypeScript](https://www.typescriptlang.org/docs/handbook/intro.html)** e no curso **[TypeScript Zero to Hero](https://github.com/glaucia86/curso-typescript-zero-to-hero)** da [Glaucia Lemos](https://github.com/glaucia86).

## 📚 Ementa

**Módulo 01 — Introdução ao TypeScript**
- O que é TypeScript e por que usar
- Configuração do ambiente
- Entendendo o `tsconfig.json`
- Hello World em TypeScript

**Módulo 02 — Tipos Básicos**
- Type Annotation
- `boolean`, `number`, `string`, `array`, `tuple`
- `enum`, `unknown`, `any`, `void`, `null`, `undefined`, `never`, `object`

**Módulo 03 — Fluxos de Controle**
- `if...else`, `switch...case`, `for`, `while`

**Módulo 04 — Funções**
- Optional, Default e Rest Parameters
- Arrow Functions e uso do `this`
- Function Overloadings

**Módulo 05 — Classes**
- Modificadores de acesso (`public`, `private`, `protected`)
- `readonly`, Getters & Setters
- Herança, métodos estáticos, classes abstratas

**Módulo 06 — Interfaces**
- Propriedades opcionais e readonly
- Function Types, Class Types
- Extensão de interfaces, uso de `Omit`

**Módulo 07 — Tipos Avançados & Novos Operadores**
- Intersection Types, Union Types
- Type Guards, Type Casting, Type Assertions
- Conditional Types, Mapped Types, `satisfies`

**Módulo 08 — Generics**
- Introdução a Generics
- Generic Types, Classes, Constraints e Interfaces

**Módulo 09 — Módulos**
- Function Modules, Default Exports
- Class Modules e aliases

**Módulo 10 — TypeScript com Node.js**
- Configuração de um projeto Node.js + Express com TypeScript

## 💻 Como Executar

```bash
# Clone o repositório
git clone https://github.com/riquegmz/typescript-studies

# Acesse a pasta do módulo desejado
cd modulo-01

# Instale as dependências (se necessário)
npm install

# Execute com ts-node
npx ts-node app.ts

# Ou compile e rode
npx tsc && node dist/app.js
```

**Requisitos:** `Node.js v18+` · `TypeScript v5+`
